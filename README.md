# Farmers-Application

A Java + JDBC + MySQL based mini project that enables direct interaction between farmers and consumers. This application allows farmers to add products, update details, and manage product listings using CRUD operations.

---

## 🚀 Features

- Add Products
- View Products
- Update Product Price
- Delete Products
- MySQL Database Integration
- JDBC Connectivity
- Exception Handling & Validation

---

## 🛠 Technologies Used

- Java
- JDBC
- MySQL
- Eclipse / STS
